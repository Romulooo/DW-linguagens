C++

C++ é uma linguagem de uso geral que permite acesso a recursos de baixo nível como ponteiros e recursos de alto nível como construção de novos tipos usando orientação a objetos e recursos de programação genérica (templates). Podemos descrever C++ por duas palavras: Elegância e performance.

Atualmente encontramos softwares elaborados a partir da linguagem de programação C++ em várias aplicações cotidianas, principalmente em jogos, por sua capacidade de resolver complexos problemas.
